开始忙碌的新一年

January 31, 2017

在这良辰鸡年，为了继续为我们的艺术家们纪录炒鸡棒的纹身作品，为了在研究的道路上大步向前， 我们将要为大家介绍一大堆新的功能。在技术方面，网站添加了一些吸引人的新功能。

首先，公告页面可以有带有自己链接的单独页面了:

  • gogo.tattoo/post/2016-11-27-vision
  • gogo.tattoo/post/2016-12-17-annual-summary

同时，纹身艺术家们的个人页面也变得越来越好看了。我们觉得鸡年是一个非常好的时间来让我们在原有的基础上优化我们的社区。 我们现在的目标是在一些链接里介绍我们的最新功能。

顶端的标签

纹身作品现在都带有标签了，所以来访者可以按照自己的标签喜好来找到并关注相应纹身艺术家， 也可以让纹身艺术家更加容易的在自己的工作历史里找到有关标签的相应纹身作品。 标签会在艺术家的个人页面顶端和纹身作品页面的底部显示。

访问gogo.tattoo的用户们可以使用这个简易的功能轻松找到他们正在寻觅的设计样式， 和擅长此类样式的艺术家。在透明化纹身样式偏好的趋势里我们因此而前进了巨大一步， 使gogo.tattoo向我们设想的方向发展。

  • gogo.tattoo/gogo/tags/dragon
  • gogo.tattoo/aid/tags/color
  • gogo.tattoo/gogo/tags/watercolor
  • //gogo/tattoo/anchored_infinity_moon

全新的纹身手稿页面

这是另一个帮助我们的艺术家与在此页面上欣赏他们作品的来访者们相连接的方法。 这是一个可以让艺术家的创作更有辨识度并让艺术家们展示他们下一个想要制作的纹身作品的手稿的地方。

  • gogo.tattoo/gogo/design

Github 代码库是可用的

对于想要加入gogo.tattoo的程序员们和寻求合作的伙伴们来说，我们的Github代码库里充满了我们网站建设中的更新和进程。 网站demo/alpha版本 里的艺术家页面现在是开源的。

Gogo-upload

Gogo-upload是用Golang写的程序。它的目的是为照片添加带有标记的水印，调整其大小并上传到IPFS。 此程序已经完成了一半了，我们邀请您来促进次程序的发展。
